# Singapore Population Demographics Visual Analytics Dashboard

## Overview
This project analyses the dynamics of Singapore’s resident population structure and geographic distribution from 2020 to 2025. Designed from the perspective of a graphical editor supporting a general-audience media article, the Tableau Story presents clear and engaging visualisations on population trends, age-sex structure, dependency ratios, geographic distribution, and dwelling-type patterns.

**Tableau Public Dashboard:**  
https://public.tableau.com/app/profile/hann.danish7698/viz/HannDanishAssignment2/Story1

## Problem Statement
The objective was to design a concise set of visualisations to support an article on Singapore’s demographic changes since 2020. The dashboard needed to communicate high-level trends, age and gender dynamics, and geographic population distribution in a way that is accessible to a general audience.

## Dataset
The project uses Singapore resident population data from the Singapore Department of Statistics, covering planning area, subzone, age group, sex, and type of dwelling.

The analysis focuses on resident population patterns from 2020 to 2025.

The Tableau Story includes six main dashboards:

2. Age-sex breakdown  
3. Age group trends  
4. Dependency ratio analysis  
5. Geographic population distribution  
6. Property / dwelling type trends  

## Key Insights
- Singapore’s resident population in households fell from 2020 to 2021, before increasing again from 2021 onwards.
- The elderly population grew faster than other age groups, highlighting Singapore’s ageing population challenge.
- Youth dependency decreased, while elderly dependency increased, causing the overall dependency ratio to rise.
- Middle-aged adults formed the largest population segment, while teens formed the smallest segment.
- There were consistently more females than males across the period analysed.
- Bedok, Jurong West, and Tampines were among the most populated planning areas.
- Public housing remained the dominant dwelling type, with medium-sized HDB flats forming a major share of resident housing.

## Visual Analytics Techniques Used
- Bar charts for high-level population trends
- Population pyramid for age-sex demographic structure
- Stacked area charts for age group composition over time
- Percentage change line charts for year-on-year demographic shifts
- Dependency ratio calculations
- Choropleth map for planning area population distribution
- Interactive filters and legends for general-audience exploration
- Consistent colour encoding across related charts
